# Data Types

## Primitives
* Number
* String
* Boolean
* Null
* Object

## Non-Primitives
* Function
* Array
* Objects

Note that `===` on non-primitives compares memory addresses.

## Checking Data Types

### Arrays

```javascript
Array.isArray(foo)
```

### Objects

```javascript
typeof foo === 'object' && foo !== null
```
